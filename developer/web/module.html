How do I add it to my home page?

1. Download the folder

2. Extract the folder (if it isn't already)

3. Copy & Paste the folder into the \Public\ branch (OfflineWeb\client\Public)

4. Create a button under the "My Links" section, or equivilent.

5. Link button to HTML file

And congratulations, you are finnished!

For developers & debug modules, make sure to go to the \test\ branch of the source code.

(OfflineWeb\client\test)

.

How do I publish my web module?

Publishing a web module isn't too hard. All you need is:

1. Your website source code

2. A Github account

3. An internet connection

.

Now let us begin

1. Get your folder with your files ready

2. Go onto the repository page (https://github.com/AWonkeyTortila/WebModuleManifest)

3. Create a branch

4. Navigate to the correct folder in the repo, and place your folder in their

5. Submit the branch in

6. Wait for it to be aproved!

Easy-peasy!
